## What is Listy?
* Listy is a web app that allows people to make their own checklists and share it with any other member (Listlet) of Listy if they choose.


## Possible uses: 

* Grocery List
* Homework List
* Dorm List

## Target Audience:
* Families
* Students
* Coworkers
